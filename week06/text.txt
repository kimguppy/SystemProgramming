Basic Navigation
You can move around using the h, j, k, and l keys.

h: move left
j: move down
k: move up
l: move right
Text Manipulation
VI has many commands for manipulating text:

dd: delete current line
yy: yank (copy) current line
p: paste after cursor
u: undo last change
i: enter insert mode
Esc: return to command mode
Advanced Features
VI offers search capabilities with the / command followed by your search term.
You can perform replacements using the :%s/old/new/g syntax.
